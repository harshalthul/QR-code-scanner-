# QR Code Scanner

A simple QR code scanner application built with JavaScript/HTML. This project demonstrates how to capture video from a webcam, detect QR codes in real-time, and display the decoded information.

## Features
- Live video capture from the device camera
- Real‑time QR code detection using the `jsQR` library
- Displays decoded text instantly
- Simple, lightweight UI

## Installation
```bash
# Clone the repository (if you haven't already)
git clone https://github.com/harshalthul/QR-code-scanner-.git
cd QR-code-scanner-
# Open `index.html` in a web browser
# On Windows, double‑click the file; on macOS/Linux, you can use `open index.html`
```

## Usage
Open the app in a modern browser, allow camera access, and point the camera at a QR code. The decoded text will appear below the video feed.
